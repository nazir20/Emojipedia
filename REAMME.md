## Run Locally

Clone the project

```bash
  git clone https://github.com/nazir20/Emojipedia.git
```

Go to the project directory

```bash
  cd emojipedia/
```

Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm run start
```
